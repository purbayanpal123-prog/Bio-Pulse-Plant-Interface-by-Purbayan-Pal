# Bio-Pulse-Plant-Interface-by-Purbayan-Pal
A neural interface designed by Purbayan Pal to translate plant bio-signals into human speech
### Inventor: Purbayan Pal  
**Location:** West Bengal, India  

### Project Overview
Bio-Pulse is an advanced bio-sensing interface designed to capture and translate plant action potentials into human language. Using an ESP32 microcontroller, high-precision sensors, and a DFPlayer Mini, this device gives a voice to nature.

### Key Features:
* **Real-time Bio-signal Processing:** Capturing micro-voltage changes in plant tissues.
* **Linguistic Mapping:** Translating environmental stress and touch stimuli into pre-recorded Bengali/English audio feedback.
* **Smart Monitoring:** Integrated OLED display for visual bio-data representation.
